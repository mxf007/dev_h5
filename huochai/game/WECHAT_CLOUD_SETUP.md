# 微信云开发配置指南

## 1. 开通微信云开发

1. 登录微信公众平台，进入小程序管理后台
2. 在「开发」-「开发设置」中找到「云开发」，开通云开发服务
3. 获取云开发环境 ID

## 2. 配置项目

### 2.1 修改环境 ID

在 [src/services/WechatCloudService.ts](file:///g:/dev_h5/huochai/game/src/services/WechatCloudService.ts) 文件中，将 `your-cloud-env` 替换为你的云开发环境 ID：

```typescript
wx.cloud.init({
    env: '你的云开发环境ID'  // 替换为你的实际环境ID
});
```

### 2.2 创建云开发目录结构

在小程序开发者工具中，右键项目根目录，选择「新建 Cloud Functions Directory」，创建云函数目录。

## 3. 创建数据库集合

在微信云开发控制台中，创建以下集合：

- `game_data`: 存储玩家的游戏数据

## 4. 部署说明

1. 确保在微信开发者工具中勾选「将 JS 编译成 ES5」
2. 上传代码时，确保包含云开发相关文件
3. 在小程序管理后台提交审核时，注意勾选云开发相关权限

## 5. 测试

1. 在微信开发者工具中测试云开发功能
2. 确保数据能够正常上传和下载
3. 检查错误处理是否正常

## 注意事项

- 仅在微信环境中启用云开发功能
- 非微信环境会自动降级使用本地存储
- 数据上传有频率限制，请合理安排上传时机
- 遵循微信云开发的使用规范和限制
